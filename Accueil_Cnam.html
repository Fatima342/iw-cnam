Module développement Web - HTML/CSS

Les deux langages incontournables dans la création d'un site internet

    Découverte et apprentissage des deux langages
  • Le Html :

    • Le HTML ou Hypertext Markup Language n’est pas un langage web de programmation. Il s’agit plutôt d’un langage de balisage du contenu d’une page sur un site internet. Les balises spécifiques utilisées dans le langage HTML permettent d’inclure certains fichiers tels que les feuilles de style, les images ou les fichiers JavaScript. Ainsi, le HTML a pour fonctionnalité principale la création de liens entre deux pages sur le web : les liens hypertextes. Aussi, le HTML permet également de donner une forme générale au texte qui sera affiché sur la page de votre site web.

  • Le Css :

    • À l’instar du langage HTML, le CSS ou feuille de style en cascade facilite la mise en forme de votre page web. Ce langage front-end permet en effet de sélectionner un élément sur la page du site et d’y appliquer une certaine mise en forme. Vous pouvez par exemple avoir recours à ce langage pour définir la couleur et la taille de la police d’écriture ou encore de l’arrière-plan de votre site. Le CSS ou Cascading Style Sheets vous permet également de dimensionner un bloc de votre page web. En d’autres termes, le CSS définit l’apparence de votre site web.
